# KUBID

KUBID는 고려대학교 학생들을 위한 경매형 중고 거래 서비스입니다.
하한가와 상한가를 설정해서 상품을 등록하면 경매가 시작됩니다. 구매자들의 입찰이 진행되고 판매자는 원하는 가격에 낙찰할 수 있습니다.
